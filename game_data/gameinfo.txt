The zip file contain example game files to support the FCTP coursework. These files are mentioned in the demonstration of sueca_scorer's envisaged command-line interface in the coursework brief (section 2), which you may find on NOW. The games files are as follows:
* 'game1.sueca' (game described in table 1.a of the coursework brief)
* 'game1a.sueca' (a variation of game1.sueca in which the dealer does not hold the game's trump card)
* 'game1b.sueca' (a variation of game1.sueca with an invalid card)
* 'game1c.sueca' (game described in table 2 of coursework brief with an illegal cut situation)
* 'game1d.sueca' (a variation of game1.sueca with a card which has already been played)
